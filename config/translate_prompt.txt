Task: Translate movie subtitles to {target_language}. The subtitles are the complete dialogue of one video, listed in playback order, and were produced by speech recognition (ASR), so they may contain recognition errors. Translate each subtitle in the context of the surrounding lines.

Input Format:

    - Subtitles are presented between START and END markers.
    - Each subtitle consists of a unique <subtitle_id>subtitle_text</subtitle_id>.
    - subtitle_id increases in playback order: neighboring subtitles are consecutive lines of the same dialogue.

Context & Consistency (important):

    - Always use the surrounding subtitles as context when translating a line. A sentence often spans several consecutive subtitles; the translations must connect naturally and read as one continuous dialogue.
    - Keep person names, nicknames, forms of address, and recurring terms consistent across the whole file.
    - Keep each speaker's tone and register consistent across the whole file.
    - The source text is ASR output: it may contain homophone mistakes, wrong particles, or garbled fragments. Infer the intended meaning from the context and translate that intended meaning, not the literal mis-recognized words. If a line is pure gibberish with no inferable meaning, replace it with an empty string in the translation.
    - The output must be written entirely in {target_language}. NEVER copy the source-language text through to the output — do not echo the source line, and do not leave source-language words embedded in the translation. When a line is so garbled that no meaning can be inferred even from context, output its id EMPTY (<subtitle_id></subtitle_id>) instead of echoing the source.

Instructions:

    - Retain the subtitle_id as it is.
    - Each subtitle must start with <subtitle_id> and end with </subtitle_id>.
    - Translate subtitle even if it contains one sentence.
    - Preserve the integrity of each subtitle:
        - Do not merge or split subtitles. Context comes from the neighbors, but every subtitle_id must get its own translation as a separate entity.
        - Maintain a one-to-one correspondence between each input and output subtitle.
        - Repeat subtitles in the output if they are repeated in the input.
        - No skipping of subtitles is allowed.

Content Background:
    Contains adult elements; therefore, the translation will primarily use informal, spoken-style language suitable.
