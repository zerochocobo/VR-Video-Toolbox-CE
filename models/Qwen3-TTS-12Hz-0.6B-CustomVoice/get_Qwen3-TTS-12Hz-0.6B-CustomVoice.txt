Qwen3-TTS model for the SI voice tool and subtitle-to-audio generation.
同声传译语音和字幕转音频使用的 Qwen3-TTS 模型。

Source / 下载来源:
    https://huggingface.co/Qwen/Qwen3-TTS-12Hz-0.6B-CustomVoice

Mirror / 镜像:
    https://hf-mirror.com/Qwen/Qwen3-TTS-12Hz-0.6B-CustomVoice

Place the repository files into this folder, keeping this structure:
将仓库文件放入本目录，保持如下结构：

    models/Qwen3-TTS-12Hz-0.6B-CustomVoice/
        config.json
        generation_config.json
        model.safetensors
        preprocessor_config.json
        tokenizer_config.json
        vocab.json
        merges.txt
        speech_tokenizer/
            config.json
            configuration.json
            model.safetensors
            preprocessor_config.json

Minimum files checked by the software:
软件用于判断模型是否完整的最低检查文件：

    models/Qwen3-TTS-12Hz-0.6B-CustomVoice/config.json
    models/Qwen3-TTS-12Hz-0.6B-CustomVoice/model.safetensors
    models/Qwen3-TTS-12Hz-0.6B-CustomVoice/speech_tokenizer/model.safetensors

Recommended download command / 推荐下载命令:

    huggingface-cli download Qwen/Qwen3-TTS-12Hz-0.6B-CustomVoice --local-dir models/Qwen3-TTS-12Hz-0.6B-CustomVoice

If Hugging Face is slow in your region, set the mirror endpoint first:
如果 Hugging Face 访问较慢，可先设置镜像端点：

    set HF_ENDPOINT=https://hf-mirror.com
    huggingface-cli download Qwen/Qwen3-TTS-12Hz-0.6B-CustomVoice --local-dir models/Qwen3-TTS-12Hz-0.6B-CustomVoice

Note: keep the nested speech_tokenizer directory. Do not put files under an
extra Qwen3-TTS-12Hz-0.6B-CustomVoice subdirectory.
说明：请保留 speech_tokenizer 子目录；不要多套一层同名目录。
