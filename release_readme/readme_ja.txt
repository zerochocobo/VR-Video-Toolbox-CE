◆ 手順1：必要な外部ツールをダウンロードしてインストール
════════════════════════════════════════════════════════

このプログラムを使用するには、先に以下の外部ツールを用意してください。

┌─────────────────────────────────────────────────────┐
│  ツール1：FFmpeg（動画処理の基本ツール、必須）            │
└─────────────────────────────────────────────────────┘

1. 次のURLを開きます：https://www.gyan.dev/ffmpeg/builds/
2. 「ffmpeg-release-full.7z」（または .zip）を探してダウンロードします。
3. ダウンロード後に解凍し、解凍したフォルダ内の「bin」サブフォルダを開きます。
   中には ffmpeg.exe、ffprobe.exe などのファイルがあります。
4. ffmpeg.exe と ffprobe.exe を使えるようにする方法は、主に3つあります。
   方法1：この2つの .exe ファイルを VR_Video_Toolbox.exe と同じフォルダへコピーする
   方法2：この2つの .exe ファイルを C:\Windows または C:\Windows\System32 へコピーする
   方法3：解凍後、「bin」フォルダを Windows の PATH 環境変数に追加する

┌─────────────────────────────────────────────────────┐
│  ツール2：Lada-CLI（AIモザイク除去モデル、NVIDIA GPU推奨）│
└─────────────────────────────────────────────────────┘

1. 次のURLを開きます：https://codeberg.org/ladaapp/lada/releases
2. ページの案内に従って最新版をダウンロードします。
3. lada-cli.exe を使えるようにする方法は、主に2つあります。
   方法1：lada-cli フォルダ全体（_internal フォルダを含む）を本プログラムと同じ階層へコピーし、
          lada-cli.exe と VR_Video_Toolbox.exe が同じフォルダにある状態にする
   方法2：解凍後、lada-cli のフォルダを Windows の PATH に追加する
          例：D:\Lada\
4. 注意：Lada は NVIDIA GPU が必要です。GPU性能が高いほど処理は速くなります。
   CPU処理は非常に遅いため推奨しません。

┌─────────────────────────────────────────────────────┐
│  ツール2b（推奨代替）：Jasna-CLI                        │
└─────────────────────────────────────────────────────┘

Jasna は Lada の新しい保守版ブランチで、機能は同等で、既存の処理手順と互換性があります。

1. 次のURLを開きます：https://github.com/Kruk2/jasna/releases
2. 最新の Windows 版をダウンロードして解凍します。
3. jasna-cli.exe を使えるようにする方法は、主に2つあります。
   方法1：jasna-cli フォルダ全体（_internal フォルダがある場合はそれも含む）を
          本プログラムと同じ階層へコピーする
   方法2：解凍後、Jasna のフォルダを Windows の PATH に追加する
          例：D:\Jasna\
4. メイン画面の「AIエンジン」で「Jasna」または「Lada」を選択できます。
   選択内容は自動的に保存されます。
5. 注意：Jasna も Lada と同様に NVIDIA GPU が必要です。


◆ 手順2：モザイク除去モードを選択
════════════════════════════════════════════════════════

プログラムを起動すると、メイン画面に複数のモザイク除去モードが表示されます。
以下を参考に選択してください。

┌─────────────────────────────────────────────────────┐
│  【推奨】ワンクリックモード                              │
└─────────────────────────────────────────────────────┘
  対応しやすいメーカー：S1VR、MDVR、VRKM、IPVR など、多くの主流メーカー。
  操作が最も簡単で、モザイク位置を自動判定して処理します。

  ▶ このモードが適しているか確認する方法
    VR機器で元動画を開き、画面下部のモザイク形状を確認します。
    モザイクが【扇形 / 弧状】に見える場合 → そのまま「ワンクリックモード」を使用します。
    モザイクが【正方形】に見える場合     → 設定で「先に魚眼視点へ変換」を有効にします。

┌─────────────────────────────────────────────────────┐
│  選択範囲 直接クロップモード                            │
└─────────────────────────────────────────────────────┘
  対象：通常の平面動画、または普通の画面で見たときにモザイクが矩形に見える動画。
  操作：モザイク範囲を手動で選択 → プログラムが処理し、元の位置へ重ね戻します。

┌─────────────────────────────────────────────────────┐
│  選択範囲 VR→平面モード                                │
└─────────────────────────────────────────────────────┘
  対象：FSVSS、SAVR、URVRSP、CRVR、PXVR など、および一部の 3DSVR 作品。

  ▶ このモードが適しているか確認する方法
    VR機器では画面下部のモザイクが【正方形】に見えるが、
    PCで元ファイルを見るとモザイクが【台形 / 斜め形状】に見える場合は、このモードを使用します。
  操作は少し複雑です。まずはワンクリックモードに慣れてから試すことを推奨します。

どのモードを使えばよいか分からない場合：
→ メイン画面の「モザイクの種類が不明なら 拡大ツールで確認」リンクを使って判断してください。


◆ 手順3：字幕を一括生成（任意）
════════════════════════════════════════════════════════

メイン画面の「JAV字幕ツール」から字幕生成機能を開けます。

この機能はモザイク除去とは独立しており、Lada は不要です。ただし、以下が必要です。
  1. FFmpeg（上記と同じ）
  2. 音声認識モデルファイル（初回使用時に「モデルをダウンロード」を押すと models フォルダへ保存されます）
  3. 【任意】NVIDIA GPU による高速化：
     CUDA 環境が検出されると自動的に有効になり、CPUより大幅に速くなります。
     NVIDIA GPU がない場合は CPU 認識に自動で切り替わります。速度は遅いですが使用できます。


◆ よくある質問
════════════════════════════════════════════════════════

Q: 「ffmpeg が見つかりません」と表示されます。
A: ffmpeg.exe が VR_Video_Toolbox.exe と同じフォルダにあるか確認してください。
   または、ffmpeg の bin フォルダを Windows の PATH 環境変数へ追加してください。

Q: モザイク除去の効果が弱い、または効果がありません。
A: Jasna と Lada は矩形モザイクに最も適しています。一部メーカーの特殊なモザイク形状は、
   AIで復元できない場合があります。これはモデル自体の技術的制限です。
   メイン画面の「AIエンジン」でエンジンを切り替えるか、別のモードを試してください。

Q: 字幕ツールの認識結果が文字化けする、または日本語ではありません。
A: このツールは標準で日本語音声を認識します。動画自体に日本語音声が含まれているか確認してください。
   認識結果は元動画と同じフォルダに .jp.srt 形式で保存されます。

Q: 実行中に "Option 'ad' not found" や "Error initializing filter" などのエラーが出ます。
A: ffmpeg のバージョンが古い、または必要な機能が省かれた軽量版を使っている可能性があります。
   本プログラムは高度な音声ノイズ低減や VR 全景フィルタを使用します。
   公式推奨ページから最新版の Full Build に置き換えてください。バージョン 6.0 以上を推奨します。
   ダウンロード先：https://www.gyan.dev/ffmpeg/builds/（ffmpeg-release-full.7z を選択）

Q: このプログラムの最新版はどこで入手できますか？
A: ホームページ：https://github.com/zerochocobo/VR_VIDEO_TOOLBOX_CE
